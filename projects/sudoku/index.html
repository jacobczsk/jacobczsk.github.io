OpenSudokuSolver

A simple sudoku solver written in Python. Licensed under Apache License. There are some example sudoku in the collection folder (easy, medium, hard, Escargot).

GitHub repo

Documentation

solve(board: list) -> (list, float) | bool

The main function. Solves the board (2D list). If it is solvable, then return the board and time it took to solve. If it isn't, then return False.

create_board(filename: str) -> list

Parse a csv file and return a board

find_empty_cell(board: list) -> (int, int) | NoneType

Get first empty cell in the board and return it's position. If there isn't any, return None.

is_valid(board: list, pos: tuple, num: int) -> bool

Check if num can be placed on pos.

Examples

import src.OpenSudokuSolver as oss # import OSS
from pprint import pprint # import prettyprint

solved, time = oss.solve(oss.create_board("collection/AI_Escargot.osud")) # load Escargot and solve it
print(f"Solved in {time} ms") # print the time
pprint(solved) # print the solved board

This example uses the create_board function. It loads the sudoku from a file.

import src.OpenSudokuSolver as oss # import OSS

    [1, None, None, None, None, 7, None, 9, None],
    [None, 3, None, None, 2, None, None, None, 8],
    [None, None, 9, 6, None, None, 5, None, None],

    [None, 4, None, None, None, None, None, None, 7],
